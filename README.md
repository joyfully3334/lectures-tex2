## Технические сведения

Компилируется при помощи `LuaTeX`.

Затем используется файл `prepare_pdf.sh`, который чистит мета-данные и сжимает
файл без серьезных потерь качества. Если есть необходимость сделать файлы еще
меньше, то можно использовать `clean_pdf.sh` с параметром 1.
