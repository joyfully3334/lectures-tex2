## Компиляция

Чтобы самостоятельно скомпилировать pdf файлы из исходников:

1. Убедитесь, что у вас установлен `latexmk` с поддержкой `lualatex`
2. Запустите файл `compile_and_prepare.sh` (в папке `scripts`) через терминал
3. Скомпилированные файлы будут находиться в папке `pdf`
